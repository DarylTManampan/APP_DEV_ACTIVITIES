# BOOTSTRAP Final Project 
**For the course Application Development**


# **[DTBlogs - Blog](https://daprojects.infinityfree.me/Blog)**

This project is a simple, static blog site developed as an educational exercise following a video tutorial by the professor. It is constructed with a collection of PHP files that utilize the **Bootstrap** framework to achieve a basic, responsive blog layout.

### Features

* Static Pages: Each blog post and main page is a separate, static php file.
* Navigation Bar: A fixed navigation bar at the top, linking to the main pages.
* Simple Search Functionality: The index file includes a basic JavaScript function to filter posts based on title and lead content as you type in the search bar.
* Contact Form: A basic, non-functional contact form (no backend processing is implemented).

---

### File Descriptions

| File Name | Description |
| :--- | :--- |
| `index.php` | The main **homepage** of the blog, listing all the blogpost snippets. |
| `contact.php` | The **Contact Us** page with sample contact information and a form. |
| `about.php` | The **About Us** page. |
| `>posts ` | Contains the `post_1.php  - post_5.php` for the static blog posts

---

This project serves as a learning exercise to understand the fundamental structure of a basic blog, the integration of a CSS framework like Bootstrap, and simple client-side interactions using JavaScript.  

*This is a direct implementation of the material taught in the video tutorial provided by the professor.*
